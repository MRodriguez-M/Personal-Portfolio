IP Address Tracker

A vanilla JavaScript, HTML, and CSS project which takes IP addresses using the IP Geolocation API and displays the location on a map. One contribution I made to the project included a bug fix that made a background image in the UI visible. I also used the API to get the user's IP information and display it as the default on load. Working on this project was my first real contribution to open-source, it was a great introduction to collaborating with other devs and working with code other than my own.

Chrome Landing Page

Open-source project that uses vanilla JavaScript, HTML, and CSS to create a custom landing page for use with the Chrome browser. I implemented functionality by adding buttons that when clicked swap the time format or search browser settings based on user preference, increasing the page's customizability. I also styled said buttons using CSS to match the mockup provided. I am very happy with my contributions to this project because getting to add features to a page felt like the next natural step to the minor bug fixes I had done up until that point.

Web Games

This project uses vanilla JavaScript, HTML, and CSS to provide a diverse collection of web-based games. I implemented the functionality and designed the color scheme for a full dark-mode feature for the main page of the site. I contributed to the project with several bug fixes, including: adjusting the end condition function for a game so that the correct outcome screen is displayed based on end-game results, managing the click function for a game's restart button so that the game resets the score to zero whenever the game restarts, adding conditional statements to a click event so that the game's score is only affected when clicking a button, and editing CSS styling and script logic for an element so that the correct/incorrect answers are highlighted in a quiz game.
