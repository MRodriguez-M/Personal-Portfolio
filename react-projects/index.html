Random Quote Generator

An app that displays a random quote and changes the displayed quote at the click of a button. The State Hook is used to help generate a random index number that determines which quote from an array will be displayed on screen. A handle click function not only displays a quote at the click of a designated button, but it also ensures that each quote is cycled through before repeating them. This was a fun one for me as it allowed me to pratice the basics of programming with React while expressing my creative side through designing a minimalist page and selecting some of my favorite quotes.
